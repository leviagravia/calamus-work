# Calamus Work

This repository is the working area for Calamus.

It is intentionally separate from any existing local directory named `calamus`.

Purpose:

- work on Calamus as Calamus
- avoid Centurio-style rename/fork work
- preserve the installed Calamus package while using this repository for controlled development
- keep changes explicit, tested, and documented

Current principle:

```text
Calamus stays Calamus.
No unnecessary rebranding.
No architectural divergence without a concrete reason.
```
