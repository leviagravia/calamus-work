# Calamus Phase 5.5.1 — Micro UX improvements

- Clip Collection: double-click or Enter inserts selected clip.
- Clip Collection: Ctrl+Alt+1..9 inserts clips 1 through 9.
- Edit: Paste as Plain Text (Ctrl+Shift+V).
- Edit: Duplicate Line / Selection (Ctrl+D).
- Edit: Move Line Up/Down (Alt+Up / Alt+Down).
- Options: Trim trailing spaces on save.
- Favourites: Add to Favourites shortcut moved to Ctrl+Alt+B to avoid conflict with Duplicate Line.
- Clip buttons refined a little further.
- No new dependencies.
