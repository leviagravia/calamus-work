# Calamus Phase 5.5 — UX Polish

Changes:
- Clip Collection buttons reduced and prevented from expanding.
- Edit menu simplified: Find/Replace is now the single visible entry; existing shortcuts still work.
- Added Options > Theme > Theme Selection with six lightweight writing themes.
- Added Options > Opacity submenu with fixed 30–100% values and an Opacity Selection dialog.
- Document Statistics opens a dedicated large information dialog.
- Added Tools > Language dialog for installed Hunspell dictionaries, with guidance for installing missing dictionaries via the system package manager.
- Added Tools > System Info.
- Selftest no longer treats Python runtime __pycache__ created after installation as a packaged-file failure.

No heavyweight dependencies were added.
